About Portfolio Manager

A small full-stack reference app: FastAPI, PostgreSQL, Nginx, and Docker.

What you can do

Use the dashboard to browse seeded portfolios, expand investments, and add or edit records. Everything persists in PostgreSQL. Developers can use the REST API with the same data model.

Stack

Backend

FastAPI · Python · SQLAlchemy

Database

PostgreSQL · init SQL seed

Frontend

Jinja2 pages · CSS · JavaScript

Ops

Docker Compose · Nginx reverse proxy

Navigation

Start at the home page for an overview, then open the dashboard for data. Use the top navigation on any screen size; on small viewports, open the menu with the menu button.
